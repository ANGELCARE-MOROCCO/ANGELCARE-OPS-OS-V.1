# Acceptance gate

MZ06 is accepted only when:
1. Exactly 300 Golden commands exist.
2. Each of the 12 families contains exactly 25.
3. Codes, names, purposes and functional fingerprints are unique.
4. All commands expose exact input/output schemas and context contracts.
5. No command allows email, WhatsApp, discount, contract, payment, capacity or price commitment.
6. Every command has 12 contract evaluation classes.
7. 3,600 contract evaluations and 30,000 benchmarks pass.
8. MZ01–MZ05 cumulative regressions pass.
9. Migration is additive, RLS-enabled and service-only.
10. ZIP, installer, checksum and rollback exist.
