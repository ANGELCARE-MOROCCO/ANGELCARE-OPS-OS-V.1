# Security and governance

- Shadow mode is mandatory.
- External actions remain disabled.
- All protected actions require command, actor, tenant, tool and approval checks.
- Database access is service-role only with RLS enabled.
- Facts, hypotheses, stale data and contradictions remain separated.
- Commands never invent price, margin, capacity, customer intent, authority, contract, discount or payment status.
- All command versions and decisions remain auditable and reversible.
