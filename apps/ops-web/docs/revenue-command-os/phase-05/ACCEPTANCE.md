# Acceptance gate

Given a commercial situation, the router must select eligible commands, exclude irrelevant commands, block unsafe commands, produce a deterministic structured run plan and perform no prohibited action.

Required proofs: manual, scheduled, event-driven, condition-driven, chained, fallback, approval-gated, rejected and simulation-only execution.

- 70 000 cas d’évaluation déterministes et adversariaux sont inclus.
