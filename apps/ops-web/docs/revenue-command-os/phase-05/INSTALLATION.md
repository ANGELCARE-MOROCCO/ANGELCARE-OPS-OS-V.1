# Installation

1. Run the surgical installer from Downloads.
2. Review and manually apply `apps/ops-web/supabase/migrations/20260720_revenue_command_os_phase5_command_kernel.sql` in Supabase SQL Editor.
3. Run the targeted typecheck and static verification scripts printed by the installer.
4. Start development and open `/revenue-command-os/command-kernel`.

The installer never stages Git, commits Git or applies the database migration automatically.
