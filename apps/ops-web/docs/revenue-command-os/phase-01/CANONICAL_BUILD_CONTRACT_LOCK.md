Phase 1 is the isolated product foundation. It delivers the shell, workspaces, permissions, feature flags, status language, environment posture, event identifiers, event outbox contract, audit contract, search, objective registry, error handling, recovery states, documentation and verification.

No later phase may delete, downgrade, rename or bypass this foundation without a versioned migration and explicit executive approval.
