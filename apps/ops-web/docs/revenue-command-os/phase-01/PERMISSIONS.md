# Permission Registry

| Permission | Purpose | Risk |
|---|---|---|
| `revenue_os.view` | Open the Revenue OS cockpit and general workspaces | Low |
| `revenue_os.manage` | Administer the module | Restricted |
| `revenue_os.objectives.manage` | Create and govern strategic objectives | Controlled |
| `revenue_os.strategy.manage` | Govern future strategy versions | Restricted |
| `revenue_os.commands.manage` | Govern future command library | Restricted |
| `revenue_os.approvals.manage` | Approve sensitive actions | Restricted |
| `revenue_os.audit.view` | Inspect Revenue OS audit | Controlled |
| `revenue_os.settings.manage` | Change allowed settings and flags | Restricted |

`revenue.view` remains a compatibility permission during the transition. It is not the target long-term authority model.
