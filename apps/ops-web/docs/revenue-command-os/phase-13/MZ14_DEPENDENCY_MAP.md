# MZ14 dependency map

MZ14 must consume only MZ13 blueprints where `ready_for_mz14=true`. It must use controlled adapters for B2B Partnerships, TrainingHub Commercial, Email OS, Gmail, WhatsApp, Calendar, opportunities, meetings, proposals, payments, Academy delivery, reporting and internal tasks. Every adapter must preserve source attribution, idempotency, approval gates, retries, failure recovery and rollback.
