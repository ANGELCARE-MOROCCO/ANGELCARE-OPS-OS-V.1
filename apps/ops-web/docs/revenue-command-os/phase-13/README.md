# MZ13 — Strategy-to-Mission Compiler

MZ13 compiles an exact, immutable, MZ12-approved strategy version into a complete internal operating package. It generates Revenue Plays, Programs, Campaigns, Waves, Account Plans, Missions, Tasks, Steps, Scripts, Evidence, Deadlines, Owners, Dependencies, Approval Gates, KPIs, Retry Rules, Rescue Routes, Stop Conditions and Escalations. External propagation remains disabled until MZ14.
