# MZ13 Dependency Map

MZ13 may consume only a strategy version with:

- MZ11 classification ready for executive review;
- final MZ12 approval record;
- all mandatory decisions present;
- all blocking conditions satisfied or formally waived;
- no expired or revoked approval;
- immutable strategy version and source trace;
- compilation preview and approval constraints.

MZ13 must not infer approval from a UI status alone.
