# Architecture

MZ07 extends the authoritative MZ05 registry and MZ06 Golden 300; it does not create a parallel command platform.

The operational layer contains 1,000 commands, versions and triggers, 334 governed schedules and 94 acyclic graphs. `revenue_os_command_library_active` is a security-invoker view that exposes exactly the Golden 300 plus MZ07 new 700. The MZ05 proof objects remain historically preserved for kernel regression but are excluded from production routing.

Coverage, semantic reviews and routing health are persisted through additive MZ07 tables. Every command remains schema-bound, permission-controlled, Shadow-safe, versioned, tested and auditable.
