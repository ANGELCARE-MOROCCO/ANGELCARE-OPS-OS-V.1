# Known limitations

- Strategy assembly, validation council, Strategy Studio, mission compilation, SaaS propagation and autonomous external execution belong to later Mega ZIPs.
- Runtime remains in Shadow mode.
- Database migration is intentionally manual.
- The focused cumulative package does not contain unrelated live-monorepo integration files such as the global permission and access-control registries; therefore MZ01–MZ03 verification is executed by the live installer against the user's complete monorepo, not falsely claimed inside the focused package.
- No production build was run.
