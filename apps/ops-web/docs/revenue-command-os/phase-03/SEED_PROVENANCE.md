# Seed provenance

The MZ03 seed library is a structured implementation of the signed Revenue Command OS contract and known AngelCare operating doctrine. It contains representative production-grade records, but it does not fabricate confidential prices, legal conclusions or unverified market facts.

Source labels distinguish:

- canonical contract;
- approved internal doctrine;
- operationally observed pattern;
- manually authored resource;
- imported document reference;
- unsupported or pending validation.

Records requiring AngelCare management confirmation remain draft, in review, conflicted or flagged by validation. Contract seeds are safe fallbacks and can be replaced through governed migrations and UI actions.
