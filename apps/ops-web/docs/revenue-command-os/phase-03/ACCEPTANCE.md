# Acceptance

Mega ZIP 3 is accepted only when:

- the twelve doctrine workspaces load under protected access;
- the migration creates all contracted tables, RLS and server-only grants;
- at least 45 governed doctrines and the associated evidence, scripts, cases, playbooks and restrictions are seeded idempotently;
- lifecycle transitions block invalid activation and retirement routes;
- effective doctrine can be distinguished from draft, retired and unsupported content;
- unresolved high/critical conflicts reduce readiness and block unsafe use;
- approvals, conflicts, versions and validation are traceable;
- global search can locate doctrines, assets, playbooks and conflicts;
- MZ01 and MZ02 verification remain green;
- external actions, autonomous pricing and strategy execution remain disabled.
