# Mega ZIP 4 — Live Revenue Signal Fabric

The Signal Fabric is the governed nervous system between AngelCare's live application data and the future Revenue Command intelligence layers. It converts heterogeneous records into explainable, deduplicated and permission-filtered revenue signals without giving a model unrestricted database access.

Route: `/revenue-command-os/signals`

Workspaces: Vue d’ensemble, Flux en direct, Sources & connecteurs, Santé des sources, Classification, Déduplication, Scans programmés, Snapshots de contexte, Données périmées, Abonnements, Accès & confidentialité, Validation du tissu.

The release includes 15 real-source adapters, 16 deterministic classifier rules, 20 isolated database tables, bounded scans, signed webhook ingestion, scheduler endpoint, source health, stale-data detection, context building, Digital Twin and Doctrine bridges, global search, premium UI, fixtures, verification and rollback.
