# Mega ZIP 5 - Production Hardening, Installers and Corporate Handover

Version 1.4.0 completes the cumulative ANGELCARE Desktop product. It introduces hardened logging and response headers, certificate and webview rejection, controlled SHA-256 update download, optional Ed25519 manifest verification, restart-to-installer flow, crash-loop recovery, sanitized ZIP diagnostics, signing/notarization configuration, native packaging scripts, CI release jobs and final operational documentation.

No WhatsApp conversation content, cookies, IndexedDB, authentication secrets or customer files are included in central governance or diagnostics. Trusted public installers still require ANGELCARE-owned Apple Developer ID/notarization credentials and a Windows code-signing certificate.
